# Анализатор зависимостей скриптов

(вариант 6)

## Что необходимо создать?

Скрипт, который рекурсивно анализирует шелл-скрипты в каталоге, находя:

- Используемые внешние команды (проверять наличие в `$PATH`)
- Источники других скриптов через `source` или `.`
- Переменные, объявленные в скрипте
- Необъявленные переменные (можно использовать опцию `-s` для строгой проверки)

Вывод: построить граф зависимостей между скриптами, отобразить в человекочитабельном формате.

## Инструкция по использованию

```bash
# (EN)
./shelldepend.sh --help
```

Введите в терминале команду и нажмите клавишу `Enter`.

```bash
# Проверка папки test со скриптами внутри
./shelldepend.sh ./test
```

Чтобы задать строгий режим проверки используйте параметр `--strict` или сокращенный `-s`.

```bash
# Проверка папки test со скриптами внутри
./shelldepend.sh --strict ./test
```

### Типичные проблемы

Для устранения ошибки запуска скрипта, необходимо добавить соответсвующие права:

```sh
chmod +x shelldepend.sh
```
