# Практика по курсам "Введение в Unix" и "Администрирование в Linux"

>В течение курса **"Введение в Unix"** рассматривается семество операционных систем Unix. Большое внимание уделено интерактивной оболочке ядра shell, которое позволяет взаимодействовать с системой Unix в режиме командной строки.
>Следующий курс под названием **"Администрирование в Linux"** предназначен для приобретения навыков управления и администрирования операционных систем на базе Linux. В ходе обучения студенты познакомятся с основными принципами работы Linux, его архитектурой и командной строкой.

Для выполнения практических заданий требуется создать скрипты на любой оболочке ядра shell.

Итоговые скрипты используют оболочку **bash** как более надежный и достаточно функциональный интерпертатор команд в отличие от аналогов (zsh, ksh, dash, fish и сам sh). Каждый скрипт проверен статическим анализатором **shellcheck** в подробном режиме.

Описание заданий и скриптов:

1. [Генератор паролей](https://github.com/Artemonkey/Intro_to_Unix_practice/blob/main/password_generator/)
2. [Анализатор зависимостей](https://github.com/Artemonkey/Intro_to_Unix_practice/blob/main/shell_scripts_dependencies_analizer/)
3. [HTTP сервис на systemd](https://github.com/Artemonkey/Intro_to_Unix_practice/blob/main/simple_http_service_on_daemon/)
