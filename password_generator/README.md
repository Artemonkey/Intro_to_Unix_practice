# Генератор паролей с критериями

(вариант 9)

## Что необходимо создать?

Скрипт, который генерирует случайные пароли по заданным критериям:

- длина (по умолчанию 12)
- обязательное наличие цифр, заглавных, строчных букв, спецсимволов
- исключение похожих символов (l, I, 1, O, 0 и т.д.)
- (Опционально) проверка на наличие в словаре (словарь – на свое усмотрение).

## Использование

Введите в терминале команду и нажмите клавишу `Enter`.

```sh
./passgen.sh
```

ИЛИ

```bash
bash passgen.sh
```

Чтобы задать длину пароля используйте параметр `--size` или сокращенный `-s`.

```bash
# Создание пароля длиной 20 символов
bash passgen.sh --size 20 
```

### Типичные проблемы

Для устранения ошибки запуска скрипта, необходимо добавить соответсвующие права:

```sh
chmod +x passgen.sh
```
